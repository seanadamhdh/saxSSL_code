
# saxSSL code

This repository contains code that was used for building an inital interactive application for the prediction of soil properties from mid-DRIFT spectra.
A shiny app is provided that allows the loading and processing of spectra foolowed by predicting soil properties employing the core saxSSL library using MBL. 
The shiny app makes use of the following packages:

Antoine Stevens and Leornardo Ramirez-Lopez (2025). An introduction to the
  prospectr package. R package Vignette R package version 0.2.8.

  <https://CRAN.R-project.org/package=shinyjs>.
